# Audio Assets

## keyboard-typing-loop.wav

- Source: Keyboard Soundpack #1 [Typing and Single Keystrokes]
- Author: unicaegames
- URL: https://opengameart.org/content/keyboard-soundpack-1-typing-and-single-keystrokes
- License: CC0 1.0 Universal
- Original file: `Generated Typing/generated-004_medium.wav`

The source pack includes human typing, generated typing, and single keypress
sounds. Aivatar currently uses one generated medium-speed typing clip as the
looping Terminal interaction sound.

## fridge-door-open.mp3 and fridge-door-close.mp3

- Source: Fridge
- Author: gloriaeffect (Freesound)
- URL: https://pixabay.com/sound-effects/household-fridge-92159/
- License: Pixabay Content License
- Original file: `audio_e2d0ffffa2.mp3`

Aivatar splits this 7-second fridge door recording into separate one-shot open
and close sounds. The close sound plays when the fridge animation enters its
closing phase.
